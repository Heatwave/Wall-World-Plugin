# Wall World Plugin

## What this plugin can do

This plugin can adjust the camera, increase player speed, and remove the resource carry limit.

## How to use this plugin

1. Download BepInEx 5.x and extract it into the Wall World folder, at the same level as `WallWorld.exe`, for example `D:\SteamLibrary\steamapps\common\Wall World`.
2. Launch the game once, then exit. This will generate the `steamapps\common\Wall World\BepInEx\plugins` folder.
3. Build the `WallWorldPlugin.csproj` project with `dotnet build`, then copy `WallWorldPlugin.dll` to the `steamapps\common\Wall World\BepInEx\plugins\` folder.
4. Launch the game. You can press `F8` to switch the camera.
